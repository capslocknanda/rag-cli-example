# Display Overview
myViewBoard **Display** enables users to mirror their desktop wirelessly to supported Interactive Flat Panels (IFPs) and Wireless Presentation Displays (WPDs). Casting is initiated from a web browser and received by the Display app running on the IFP/WPD.
  

  

## Availability
The myViewBoard Display app turns any device into a wireless receiver that can mirror another device‚Äôs screen by providing a **Display Code** + **One-time password**. Display is available on the following platforms:
  

  * Preinstalled on selected **ViewBoard** hardware.
  * Install on any IFP or WPD that runs:
    * **Android 6.0** or higher (Display for Android)
    * **Windows** (Display for Windows)
