# TeamOne
The **TeamOne** section in Insights aggregates data related to the use of the TeamOne app. By default, the information presented is based on activity from the last 90 days.
## TeamOne sections
See the table below for a quick description of each section. 
**Active users** |  **Total user activities** Displays the number of TeamOne activities, broken down by day and by platform.  
---|---  
**Sessions** |  **Total sessions** A session is counted when at least one action is performed in a TeamOne installation. This total counts all sessions across all TeamOne installations, organized by platform.  
**Daily TeamOne sessions** Displays the number of sessions recorded, broken down by day and by platform.  
**Groups** |  **Total groups** Total number of groups in your TeamOne environment.  
**Daily active groups** Displays the number of groups that opened a board, broken down by day and by platform.
