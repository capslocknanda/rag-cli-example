# myViewBoard
The **myViewBoard** section aggregates data related to the use of the myViewBoard app. By default, the information presented is based on activity from the last 90 days.
This section is organized into several tabs. See the sections below for a quick description. 
## Trend Report
The **Trend Report** tab displays a breakdown of session data reported by the myViewBoard app for the given period. A session is counted when an action is performed in an enrolled myViewBoard installation in your entity on a Windows OS, iOS and Android device.
**Sessions** |  **Total Sessions** Total number of sessions logged for the given period, organized by platform.  
---|---  
**Daily myViewBoard sessions** Breaks down the session data by day and by platform.  
**Sign-in** |  **Daily user sign-in by platform** Displays the number of _Sign in_ actions recorded, broken down by day and by platform.  
**Tools** |  **Usage of myViewBoard tools** Summary of data related to usage of tools selected from myViewBoard's primary tool categories. Use the following icons to view more details:  
## Usage Report
The **Usage Report** tab summarizes how often users interacted with myViewBoard for the given period.
**Launches** |  **Total myViewBoard launches** Installations must be enrolled to be counted. To verify enrolled instances, navigate to the following location on the myViewBoard admin settings page:  **myViewBoard Management** Â» **Software Instances**  
---|---  
**Sign-in** |  **Daily signed in users** Total signed in unique users in myViewBoard per day. Multiple sign-ins by the same user only count once.  
**Canvas** |  **Daily canvas sessions** A session is counted when at least one action is performed in a myViewBoard app.  
**Daily canvas hours** Duration of a session on an enrolled myViewBoard installation or on myViewBoard online.  
## Individual Report
The **Individual** Report tab provides usage detail for each user within the given period.
**Users** |  **Total users** Total number of users registered with the entity  
---|---  
**Top usage by user** Review usage data for individual users to understand how your entity's users use myViewBoard.
