# AirSync sender
On supported IFPs, AirSync is pre-loaded on the system and does not require additional installation. To connect with the AirSync on the IFP, it is highly recommended for target devices to use the AirSync sender for sharing.
  

  

  

## Download AirSync sender
The AirSync Sender supports several platforms. Download the version that matches the OS of your target device:
  

  

AirSync Sender for Windows is currently incompatible with ARM architecture devices.
  

  

  

  

  

## Update AirSync Sender
You can update AirSync Sender through the following methods:
  

  * AirSync Sender has a built-in updater that automatically performs software updates.
  * Under the AirSync Sender settings, click **Check for updates**.
  * Manually check for updates on Google Play and Apple Store.


  

  

## AirSync web sender
If the AirSync Sender is not available, you can still share screens via the AirSync website. There is no need to install additional software. To share over the internet, open a web browser and connect to the following URL to start sharing: 
  

  

  

However, note the the following limitations for the web version:
  

  * No support for local area network connections.
  * No support for touch feedback. 
  * No support for receiving IFP screens.
  * No support for pen stroke functions.
  * No support for sharing in high-quality mode.


  

  

  

## AirSync PWA (Progressive Web App) sender
Install the PWA version AirSync sender if you are:  
  

  * Unable to install the Android version AirSync sender in a ChromeOS environment
  * Unable to share audio through the Android version AirSync sender on ChromeOS


  
  

After installation, you can use all web sender features without opening the browser. To install the AirSync PWA (Progressive Web App) sender, perform the following steps:  
  

  1. Launch your browser and go to the AirSync web Sender site:  

     *   
  

  2. Click the install_desktop icon on the right side of the URL bar. 
  3. Follow the instructions to complete the installation. 


  

  

  

  

  

## AirSync macOS sender independent version
Download and install the macOS Sender independent version directly onto your Mac. The independent version provides the following benefits:
  

  * Enhances the user experience and access expanded functionality.
  * Bypasses the limitations of restricted network environments that prevent installation via the Apple Store. 
  * Supports screen sharing in extended mode on IFP, with the touchback feature enabled. 


  

  

### Installation
To install the independent version, follow the steps below:
  

  1. Launch your web browser and navigate to the following site:  

     *   
  

  2. Locate and download the PKG file to your Mac device. 
  3. Once done, double-click the PKG file and follow the instructions on screen. 
  4. Wait for the extraction to complete.
