# Cloud Access Control
Use the **Cloud Access Control** page to enable apps and cloud services that will be available to the users of the entity. 
  

  

  

## How to access
To access this page:
  1. Sign in to 
  2. On the landing page, locate the **myViewBoard** tile.   
Note the ellipsis icon when you hover over.
  3. Click the **myViewBoard admin settings**.
  4. The **myViewBoard admin settings** page appears.
  5. Click **Cloud Access Control** on the side panel.


  

  

**Important**
  

  * Any changes made here only apply to your myViewBoard environment. 
  * You can only access this page if any of the following conditions are true: 
    * You have signed in under a myViewBoard owner account 
    * You have been assigned with the Admin role for the myViewBoard app

## Supported cloud services
The following cloud services are supported:
  

Icon | Service  
---|---  
| Google Drive  
| OneDrive  
| OneDrive for Business  
| SharePoint  
| Dropbox  
| Box  
| Nextcloud Files  
| Google Classroom  
  

If a service's box is checked, users will be able to connect to the service in their user settings.
  

## Enabling cloud services
To enable specific Cloud Services for your users, follow these steps:
  

  1. Sign in your entity admin account.
  2. Go to the **myViewBoard** **admin settings** page.
  3. Click **Cloud Access Control** section on the side panel.
  4. Check the box for all services that should be available for your users.   
Note that any changes to this page are automatically saved, but allow some time for the updates to take effect. 


  

Entity users will now be able to connect with the service within their myViewBoard settings.
  

By default, all services are enabled and available to entity users. To disable a cloud service, simply uncheck the box for that service.
[Delete](https://www.viewsonic.com/solution/kb/en_US/vs-myviewboard-management/vsa-apps-and-cloud-services)
**Important**
Once a cloud service is disabled, any entity user currently bound to a cloud service will automatically lose access. Kindly inform your users before disabling a service.
  

  

## Using cloud services 
For guidance on how entity users can bind to the cloud services, see the following article: 
  * [Cloud integration for myViewBoard users](https://www.viewsonic.com/solution/kb/en_US/account-settings/cloud-integration)
