# Overview: Using ClassSwift Hub 
The **ClassSwift Hub** is a web-based platform that helps teachers and administrators manage classes, monitor lesson data, and review usage insights. It connects seamlessly with the **ClassSwift App** , ensuring that all class information, student data, and quiz results are synchronized.
