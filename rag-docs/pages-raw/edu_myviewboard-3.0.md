# Bring Your Lessons to Life
## Newly Updated ViewSonic® myViewBoard® Software
**ViewSonic ® myViewBoard®** is designed to help teachers build dynamic, engaging lessons without the hassle. Streamlined tools, simplified workflows, and flexible content integration help maximize instructional time, encourage student participation, and support diverse learning needs. It’s a powerful platform to help you teach more effectively.
[Schedule Demo](https://www.viewsonic.com/edu/myviewboard)
SCHEDULE DEMO
## Intuitive Design That Works the Way You Teach
myViewBoard 3.0 features a modern interface for a consistent, seamless experience across Windows, ChromeOS, Android and Apple devices. The new Magic Box organizes tools into easy-to-navigate categories with helpful descriptions, while Long Press shortcuts on ViewBoard displays let you instantly write or access tools without breaking your flow. **Everything’s designed to be faster, smarter, and easier—so you can focus more on teaching, not troubleshooting.**
## ISTE-Certified Edtech Solution
ViewSonic myViewBoard® software has earned an official ISTE Seal. This certification confirms that **ViewSonic’s award-winning myViewBoard software supports evidence-based, high-impact pedagogy and facilitates scalable, equitable learning experiences.**
## More Formats, More Flexibility
**Open and work with a wide range of file types** —PPT, PDF, Google Slides, .iwb, .flipchart, and more. Tabbed browsing makes it easy to manage multiple lessons at once, while cloud integration and QR code sharing streamline access for students. It’s seamless multitasking, made for the classroom.
## Inclusivity Tools for Every Learner
With features like Irlen Filters, text-to-speech, and multilingual support, **myViewBoard 3.0 promotes equitable learning for all students**. It includes scientifically supported background colors and spotlight shapes, along with the most extensive variety of background styles for diverse learning needs.
## Enhanced Participate Mode
Switch effortlessly between full screen and canvas modes while tracking student progress. Teachers can upload custom backgrounds and insert interactive learning elements into the participant model. New pop-up windows allow for individual grading and progress tracking, **making it easier to monitor and assess students in real-time**.
## Customizable Pen Tools
**Easily customize your pen tools** with quick color selection, adjustable transparency, and size settings. The AI Pen allows you to hand-draw shapes that can be instantly searched or used for image insertion—eliminating the need to type. Plus, access your favorite tools with the convenient long-press function on ViewBoard displays.

## **myViewBoard for Windows**
Teaching tools should support good planning. Ideal for prepping lessons, myViewBoard for Windows unlocks the full power of a versatile digital whiteboarding tool. Teachers can quickly and easily add content, set up complete lessons, and then save those lesson plans.
  * **Full Windows and Microsoft Support:** Built to integrate with Windows and familiar tools like Word, PowerPoint, and PDF.
  * **Advanced Functionality:** Full access to all myViewBoard functions within Magic Box to prepare lessons as they will be delivered.
  * **3rd Party Integrations:** Use popular tools like OneDrive, Google Drive, and Dropbox.


## **myViewBoard for Android**
It’s important that presentation tools are both comprehensive and simple. myViewBoard for Android manages to provide teachers with a multi-media presentation and participation tool designed for both delivering planned material and adding more on the fly.
*For compatibility with ViewBoard interactive displays, please 
  * **Lite Version for Universal Use:** With few demands on the device itself, users can present on devices of all sizes.
  * **Versatile Magic Box Tools:** Teachers can get access to a wide array of digital tools for ad hoc teaching and learning.
  * **Advanced File Management:** Files are easy to save and share on popular cloud storage services like Google Drive, OneDrive, Dropbox and more.


## **myViewBoard for iOS**
Teaching tools need to work anywhere, so myViewBoard also works on any iOS device and MacBook running the M1 chipset or newer. Teachers get everything they need to prepare, present, and encourage participation on their preferred devices.
  * **Customized for iOS-capable Devices:** Users can run the same digital whiteboard on any device running off an M1 chip.
  * **Powerful Toolset for Teachers:** Teaches get easy access to all the tools they need to engage their students.
  * **Integrates with File Management Tools:** It’s easy to store teaching materials on a variety of cloud drives (Dropbox, OneDrive, etc.) for easy access anywhere.


## **myViewBoard Online**
Lesson planning takes at least as long as teaching that lesson. With the right tools, however, teachers can create the teaching materials of their dreams. Now educators can create multimedia lessons from anywhere and share them with anyone.
  * **Powerful Toolset for Teachers:** Teaches get easy access to all the tools they need to engage their students.
  * **Integrates with File Management Tools:** It’s easy to store teaching materials on a variety of cloud drives (Dropbox, OneDrive, etc.) for easy access anywhere.
