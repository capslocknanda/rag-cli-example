# Security
Update your **Security** settings to keep you account safe and secure.
  

## Access Security settings
To access your **Security** settings directly, perform the following:
  

  1. Navigate to the following URL:
     * <https://cloud.viewsonic.com>  
  

  2. Click **Sign in** on the upper right.
  3. On the next page, use your credentials to sign-in.
  4. The next screen should show your **ViewSonic Account** page.
  5. Click **Security** on the side panel.


  

  

  

**via myViewBoard.com**
To access **Security** settings while logged in 
  

  1. Sign-in to myviewboard.com.
  2. Click your user avatar on the upper right.
  3. Click **Manage account** on the dropdown.
  4. Click **Security** on the side panel.


  

  

  

## Security
This section allows users to manage security settings related to signing in to their ViewSonic account. Click on a tab on the left panel to access available options.
  

### Change password
Use this tab to update your sign-in password for ViewSonic services. 
  

If you signed up _using your email address_ and would like to change the password you set during sign up, perform the following:
  

  1. In the **Current password** input box, type your existing password. 
  2. In the **New password** input box, type your new password. 
  3. In the **Confirm new password** input box, type your new password. 
  4. Click **Confirm**.


  

Note the following behavior:
  

  * For security reasons, any text typed in the input box will render as circles. To switch the rendered input to plain text, click the 
  * Ensure that passwords provided meet the following criteria:
    * At least 8 characters.
    * At least 1 uppercase letter.
    * At least 1 lowercase letter.
    * At least 1 number.
    * At least 1 special character.


  

  

If you signed up S _ingle sign-on_(for example: using Google, Microsoft, Apple, or another supported SSO provider), and you haven't previously set a ViewSonic account password, perform the following:
  

  1. Click the **Forgot password?** link.
  2. Click **Yes** on the prompt to confirm.
  3. The system sends an email to your registered email address.
  4. Check your email and locate the new message from ViewSonic.
  5. Click the link provided in the email.
  6. The link redirects to a page where you will need to set up a new password.
  7. Follow the instructions on the page to set up a new password.


  

  

Note that the link provided in the email expires after 1 hour. If the time limit has passed, you will need to go through the same process again to resend the email.
  

To verify if the password update was successful:
  1. Launch a ViewSonic app / service / website (that requires login) on Windows, Android, or iOS.
  2. Sign in using your email address and new password.
  3. Your account and settings should load successfully.


  

  

  

### Two-factor authentication
Enable two-factor authentication (2FA) to add an extra layer of security when you sign in using your ViewSonic account. If enabled, you will need to perform the following steps every time you sign in:
  

1 Sign-in using your ViewSonic account and password.
2 Approve the sign-in using the Google Authenticator app.
  

  

This process requires the Google Authenticator app installed on your mobile device. Download the app first from the following links. Select the one that is applicable for your mobile device:
  

  

Install, set up, and familiarize yourself with the Google Authenticator app before proceeding with the steps below:
  

  1. On your **ViewSonic Account** settings page, go to:  
**Security Â» Two-factor authentication**.
  2. Click **Enable 2FA**.
  3. Type your password. 
  4. A popup containing a QR code appears, with a section that requires an input verification code.  
  

  5. To get the input verification code:
    1. Launch the Google Authenticator app.
    2. When prompted, select the **Scan a QR code** option.
    3. Scan the QR code on the ViewSonic 2FA settings page.
    4. The Google Authenticator app generates an input verification code.  
  

  6. Type this input verification code into the ViewSonic 2FA settings page.
  7. Click **Confirm**. 


  

  
  
If you signed up using Google, Microsoft or Apple single sign-on (SSO), you can use the 2FA feature supported by these services instead.
