# System requirements
Verify that you meet the minimum and recommended requirements for devices and browsers when using TeamOne.
  

## Hardware requirements
TeamOne has been tested to run optimally on devices that have the following specifications.
  

### Minimum requirements
Minimum requirements for running TeamOne on desktops and laptop PCs that run Windows and macOS operating systems are given below.
  

  
| Windows | Mac  
---|---|---  
CPU | 2.3 GHz (2 cores/4 threads),   
Intel i5 | 64-bit Intel  
RAM | 8GB | 8GB  
OS | Windows 10 | macOS 11 (Big Sur)  
  

  

TeamOne is supported on the following ViewSonic devices:
  

ViewSonic devices  
---  
IFP | IFP105S  
IFP50-5(F)  
IFP53  
IFP63  
IFP92UW  
IFP105UW  
  
  

  

For mobile devices:
  

Android | iOS | iPadOS  
---|---|---  
Android 12 | iOS14 | iPadOS14  
  

  

### Recommended requirements
For optimal performance, we recommend using TeamOne on desktop and laptop PCs with the following specifications:
  

  
| Windows | Mac  
---|---|---  
CPU | 3.5 GHz (4 cores/8 threads),   
Intel i7 | Apple M1  
RAM | 16GB (DDR4) | 16GB (DDR4)  
OS | Latest OS | Latest OS  
  

ViewSonic devices that have the following specifications can run TeamOne at optimal performance:
IFP specs  
---  
CPU | 2.8 GHz (4 cores/8 threads)  
RAM | 16GB (DDR4)  
OS | Windows 10/Android 12  
  

  

## Browser requirements
To run TeamOne on a browser, verify that it meets the following requirements:
  

### Supported browser versions
Browsers must have graphics acceleration enabled and updated to at least the following versions:
Browser | Minimum version  
---|---  
Chromium | 95  
Chrome | Latest 5 versions  
Edge | Latest 5 versions  
Safari | 17  
  

  

### Download & upload speeds
  
| Minimum | Recommended  
---|---|---  
Network | 10 Mbps download /   
10 Mbps upload | 100 Mbps download / 40 Mbps upload on a wired connection
