# Access TeamOne
To start using TeamOne, you will need the following:
  * Access to the TeamOne dashboard
  * a ViewSonic account


  

## Login via TeamOne site
The easiest method to access TeamOne is to start TeamOne via the ViewSonic website:
  

  1. Launch a web browser and go to the following site: 
     * [https://teamone.viewsonic.com](https://teamone.viewsonic.com/)  
  

  2. Click on **LOGIN**.
  3. Sign-in with your ViewSonic credentials. You can also use Single Sign-On (SSO) to sign in with a Google, Microsoft, or Apple account.  
  

  4. On first launch, you will need to perform the following:
    1. Complete a few questions to tailor the templates TeamOne shows.
    2. Select trackpad and grid style preferences.  
  

  5. Once you're done, TeamOne presents you a new empty board. You can now start collaborating.


  

  

  

## Sign up for new account
If you do not have an account, you will be redirected to the TeamOne sign up page. 
  

  1. Launch a web browser and go to the following site: 
     * [https://teamone.viewsonic.com](https://teamone.viewsonic.com/)  
  

  2. Click on **SIGN UP NOW**.
  3. On the next page, perform one of the following to sign up: 
    1. Use Single Sign-On (SSO) with a Google, Microsoft, or Apple account. 
    2. Provide your first and last name, then specify an email address and password that you will use as login credentials.  
  

  4. Click **Sign Up**. 
  5. Afterwards, you'll receive an activation email. If you signed up via Single Sign-On (SSO), no account activation is required.
  6. On first launch, you will need to perform the following:
    1. Complete a few questions to tailor the templates TeamOne shows.
    2. Select trackpad and grid style preferences.  
  

  7. Once you're done, TeamOne presents you a new empty board. You can now start collaborating.


  

  

  

## Install on your device
TeamOne also offers installers that allow you to access TeamOne locally on your device.
  

  1. Launch a web browser and go to the following site: 
     * [https://teamone.viewsonic.com](https://teamone.viewsonic.com/)  
  

  2. Scroll down to the **Get the App on Your Favorite Platform** section.
  3. Click on the correct link to download TeamOne on your target platform. The following options are available:
     * Microsoft Teams
     * Windows
     * Android
     * macOS
     * iOS
     * iPadOS  
  

  4. Alternatively, you can also go to the following site to autodetect your device and redirect you to the correct download page: 
     * <https://teamone.viewsonic.com/download>  
  

  5. Run the installer and wait for the installation to finish.
  6. Launch TeamOne. You will need to sign in to access all the features.


  

  

  

## Install as a Windows app
TeamOne is also available on the Windows Store. To install TeamOne on your Windows device, perform the following:
  

  1. Go to the Microsoft store.
  2. Search for **TeamOne**.
  3. Click **Download**.
  4. Wait for the install to finish.
  5. Click **Open** to launch TeamOne.

If you need to setup your teams and users first, you may want to go to the TeamOne dashboard to perform further customizations.
