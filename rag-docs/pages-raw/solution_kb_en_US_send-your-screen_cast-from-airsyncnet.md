# Cast from AirSync web sender
You can use the AirSync web sender site to share your screen to the IFP. The target IFP and the device can be on different networks as long as both devices have a stable internet connection.
  

## Use AirSync web sender
To share via the AirSync web sender, perform the following steps:
  

_On the target IFP:_
  1. Open the AirSync app.   
  

  2. Note the following details: 
    1. Display code 
    2. One-time password   
  

  3. Leave the app running, then head over to your device.


  

  

  

_On your device:_
  1. Launch a browser window and go to the following URL:
     *   
  

  2. Fill in the following details from the target IFP: 
    1. Display code 
    2. One-time password   
  

  
  

  3. Click **Next**. 
  4. On the dialog that appears, select the resource you want to share to the IFP: 
    1. **Tab**. Share the contents of a single tab within the current browser.
    2. **Window**. Share the contents of one application currently running. The application window must not be minimized.
    3. **Entire Screen**. Share the contents of your main display in full-screen. For multi-monitor setups, specify which display to share.   
  
  

  
  
  

  5. Click **Share**.
  6. Sharing starts instantly and your selection should now start playing on the IFP.
