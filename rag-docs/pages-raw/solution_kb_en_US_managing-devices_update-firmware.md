# Update firmware
Send the **Update firmware** command to remotely update the firmware of multiple devices. This update can be performed on demand or on schedule.
  

  

  

## Update on demand 
To send the **Update firmware** command, launch Manager and perform the following: 
  

  1. Click the 
  2. Click the 
  3. Click the **All Devices** tab.  
  

  4. On the device list, click the checkbox of the devices you want to manage.  
A sidepanel should appear on the right.
  5. On this new sidepanel, scroll down to the following option:  
  

Update Firmware
  
  

  6. Click the   
The following options appear:  
  

  
  

  7. Click **Update now** to check for updates on the ViewSonic OTA servers. If updates are available, apply them immediately.  
  

  8. Alternatively, click **Update from external source** to upload a firmware (FW) image file from an online URL to your ViewSonic device.   
  

    1. Download the latest FW image from ViewSonic's support page. Verify that this image is compatible with your device. 
    2. Upload it to a location accessible from the web.
    3. Type this location in the **URL** text box.
    4. Click **Confirm**.  
  

  9. Manager sends the command to the selected devices. 
    1. Wait for the firmware to download and install. 
    2. To verify, interact with all target devices and check if the action was applied properly.


  

  

**Warning**
The **Firmware Update** action triggers an automatic **_force-restart_** on all target devices. Send this action only if you are sure that the target devices will not be in use.
  

  

## Update on schedule 
To send the **Auto Firmware Update** command at specific dates only, launch Manager and perform the following: 
  

  1. Click the 
  2. Click the 
  3. Click the **All Devices** tab.  
  

  4. On the device list, click the checkbox of the devices you want to manage.  
A sidepanel should appear on the right.
  5. On this new sidepanel, scroll down to the following option:  
  

Auto Firmware Update
  
  

  6. Click the   
The following options appear:  
  

  

  
  

  7. Toggle **Auto Firmware Update** to ON to have Manager check regularly and automatically update selected devices immediately once new firmware is available online.  
  
  

  8. Toggle **Schedule Firmware Update** to ON to only check for updates on the specified dates.   
  

    1. Click **Edit.**
       * Set **Daily** or **Weekly**.
       * Set a **Time**.
       * Set a **Weekday**.
    2. Click **Confirm**.  
  

  9. Wait for confirmation that the firmware schedule has been set. 
  10. If new firmware is available on the scheduled date, the update process is triggered immediately. 


  

  

**Warning**
The **Auto F****irmware Update** action triggers an automatic **_force-restart_** on all target devices. Send this action only if you are sure that the target devices will not be in use.
  

  

## Troubleshoot
If the action does not work, check if the device supports the **Firmware Update** or **Auto Firmware Update** feature. For more details, refer to the list here: 
  * [Manager device - Compatibility chart](https://www.viewsonic.com/solution/kb/other-manager-info/device-capability-chart)


  

Note that firmware updates may only affect the on-board firmware of the device, and not the Android version. To update the Android version of your device, please navigate to the following section:
  

**Jobs**
