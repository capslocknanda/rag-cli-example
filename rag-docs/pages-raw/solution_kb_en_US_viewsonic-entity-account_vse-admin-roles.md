# Admin Roles
Use the Admin Roles page to assign roles to specific users. Note that you can only assign roles to users that have been already added to your entity via the Users page.
  

  

  

## Access Admin Roles page
To access the **Admin Roles** page in your entity account, perform the following:
  

  1. Navigate to the following URL:
     * <https://cloud.viewsonic.com/entity>  
  

  2. Click **Sign in** on the upper right.
  3. On the next page, use your credentials to sign-in.
  4. The next screen should show your **Entity** page.
  5. Click **Admin Roles** on the side panel.


  

  

## Admin roles
Assign roles to specific users to help you manage the entity. Roles grant users rights to perform specific actions on the entity. The following roles are available:
  

Role | Description  
---|---  
Domain Admin | Manage domain registration  
User Admin | Add or remove users  
Subscription Admin | Manage the subscriptions of your entity  
Role Admin | Manage roles to provide admin permissions to users  
  

  

## Assign Roles 
To assign these roles, login to the **ViewSonic Entity** page and perform the following:
  

  1. Click admin_panel_settings**Admin Roles** on the left panel.
  2. On the **Admin Role** table:
    1. Locate the row of the role you want to assign.
    2. On this row, click the value under the **Members** column.  

  3. On the next page, click admin_panel_settings**Assign members**.
    1. Type a few characters to start filtering the by email addresses.
    2. Select one or more target users from the dropdown.
    3. Click cancel to remove a specific user, or close to reset the list.
    4. Click **Assign** to apply your changes.  
  

  4. You target users should now be listed in the table.


  

  

## Unassign roles 
To remove users from these roles, login to the **ViewSonic Entity** page and perform the following:
  

  1. Click admin_panel_settings**Admin Roles** on the left panel.
  2. On the **Admin Role** table:
    1. Locate the row of the role you want to update.
    2. On this row, click the value under the **Members** column.  

  3. To remove a single user from the role:
    1. On the **Members** column, locate the email address of the user to remove. 
    2. Click search to filter the list, if you have a large list of users.
    3. Click delete_outline to unassign the user from this role.   
  

  4. To remove multiple users from the role.
    1. Select one or more target users from the table.
    2. Click close to reset your selection and start over again.
    3. Click admin_panel_settings**Unassign members** to apply your changes.  
  

  5. The removed users should no longer appear in the table.
