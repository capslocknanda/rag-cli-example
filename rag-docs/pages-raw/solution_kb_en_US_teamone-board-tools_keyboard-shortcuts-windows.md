# Keyboard shortcuts
TeamOne supports several keyboard shortcuts when working on Windows and macOS.
  

## General hotkeys
## Windows
## MacOS
**Actions** | **Keys**  
---|---  
Copy |  `Ctrl` + `C`  
Paste |  `Ctrl` + `V`  
Duplicate |  `Ctrl` + `D`  
Cut |  `Ctrl` + `X`  
Undo |  `Ctrl` + `Z`  
Redo |  `Ctrl` + `Shift` + `Z`  
Select All |  `Ctrl` + `A`  
Deselect | `Esc`  
Switch to selection tool | `Esc`  
Select multiple objects |  `Ctrl` or `Shift` + Left Click  
Unselect an object from selected objects |  `Ctrl` or `Shift` + Left Click  
Delete |  `Delete` or `Backspace`  
Lock / Unlock |  `Ctrl` + `Shift` + `L`  
Create a shape with the same ratio |  `Shift` + Drag  
Create a connector line |  `Ctrl` + `C`  
Send to back | `[`  
Bring to front | `]`  
Duplicate an object |  `Alt` + Drag  
**TIP**
Some keyboard have a **FnLock** key. If**FnLock** is enabled, hotkeys may not respond as expected. Please disable **FnLock** first, then try again.
