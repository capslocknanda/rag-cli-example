# Users
Use the **Users** page to add and remove users from your Entity. This page allows admins to add users via the following methods: 
  

  * Auto-add all users in your Google Workspace, Microsoft Entra ID, or Microsoft SCIM environment via SSO
  * Batch add users via CSV upload
  * Manually add individual users 


  

  

  

  

## Access Users page
To access the **Users** page in your entity account, perform the following:
  

  1. Navigate to the following URL:
     * <https://cloud.viewsonic.com/entity>  
  

  2. Click **Sign in** on the upper right.
  3. On the next page, use your credentials to sign-in.
  4. The next screen should show your **Entity** page.
  5. Click **Users** on the side panel.


  

  

## Add single user 
If you need to add a single user, login to the **ViewSonic Entity** page and perform the following:
  

  1. Click group**Users** on the left panel.
  2. Click person_add**Add new user**.  
  

  3. Provide the following details.
    1. First name
    2. Last name
    3. Email   
  

  4. Click **Add**.
  5. A successful import triggers the following:
    1. The **Success** notification appears 
    2. The system sends an email invite to the new user added


  

  

### Naming conventions
Note the following rules when adding entries: 
  

  * Only letters (A-Z, a-z), numbers (0-9), and certain symbols (_@-.) are allowed.
  * The **First Name** and **Last Name** fields must not have any spaces. 


  

  

## CSV import
If you need to add multiple users, use a CSV template. To download a sample template, go to the **ViewSonic Entity** page and perform the following:
  

  1. Click group**Users** on the left panel.
  2. Click **More options** keyboard_arrow_down.
  3. Click **Bulk upload users**.
  4. Click download**CSV Template** on the popup.
  5. Save to your local drive.


  

Afterwards, replace the values in the template with the details of the users you need to add. To upload the updated CSV file, go to the **ViewSonic Entity** page and follow these steps:
  

  1. Click group**Users** on the left panel.
  2. Click **More options** keyboard_arrow_down.
  3. Click **Bulk upload users**.
  4. Click **Choose file**.
  5. Click **Upload** to import your updated CSV file.  
  

  6. A successful import triggers the following:
    1. The **Success** notification appears 
    2. The system sends out the following: 
       * For Public entities: email invites to all users in the list.
       * For Regular and Shared entities: notification emails.


  

  

## Sync via SSO
myViewBoard offers several options for adding users via single sign-on (SSO): 
  

  * Google Workspace
  * Microsoft Entra ID
  * Microsoft Entra ID SCIM
  * ClassLink


  

Before proceeding, login as administrator of your Microsoft Entra ID (Global Administrator), Google Workspace (Super Admin), or any other account you will configure for SSO, and enable the mailbox of any user that will be added to your entity. This ensures that they can receive important updates after the process is complete.
  

  

### Google Workspace
To set up SSO for Google Workspace, follow these steps:
  

  

  1. Click group**Users** on the left panel.
  2. Click **More options** keyboard_arrow_down.
  3. Click **Sync users with Google Workspace**.
  4. On the next page, sign in with your Google Workspace Super Admin account.   
Allow permissions if necessary.  
  

  5. If login was successful: 
    1. Verify that the value for **Your domain** shows the correct target.
    2. Otherwise, click the dropdown and switch to the correct one.  
  

  6. Select one of the following sync options:  
  

     * **All users**.  
Create accounts for all users in your organization.  
  

     * **Group**.   
Create accounts for pre-defined groups in your organization.  
  

     * **Unit**.   
Create accounts for pre-configured units in your organization.  
  

     * **Keep current users**.  
If checked, previously added users will remain in the entity. This is checked by default.  
  

  

  7. Click the **Sync** button button and wait for the sync process to complete.
  8. To verify that the sync was a success:
    1. New users from your domain are now listed in the **Users** page 
    2. These new users can now use the **Google SSO** option to successfully access the ViewSonic dashboard.

### Microsoft Entra ID
To set up SSO for Microsoft Entra ID, follow these steps:
  

  

  1. Click group**Users** on the left panel.
  2. Click **More options** keyboard_arrow_down.
  3. Click **Sync users with Microsoft Entra ID**.
  4. On the next page, sign in with your Microsoft Entra ID Global administrator account. Allow permissions if necessary.  
  

  5. If login was successful: 
    1. Verify that the value for **Your tenant** shows the correct target.
    2. Otherwise, click the dropdown and switch to the correct one.  
  

  6. Select one of the following sync options:  
  

     * **All users**.  
Create accounts for all users in your organization.  
  

     * **Group**.   
Create accounts for pre-defined groups in your organization.  
  

     * **Keep current users**.  
If checked, previously added users will remain in the entity. This is checked by default.  
  

  

  7. Click the **Sync** button button and wait for the sync process to complete.
  8. To verify that the sync was a success:
    1. New users from your tenant are now listed in the **Users** page 
    2. These new users can now use the **Microsoft SSO** option to successfully access the ViewSonic dashboard.

### Microsoft SCIM
To set up SCIM, follow these steps:
  

  

  1. Click group**Users** on the left panel.
  2. Click **More options** keyboard_arrow_down.
  3. Click **Sync users with Microsoft SCIM**.
  4. Click **Download SCIM setup instructions.**
  5. The document provides steps on how to configure Microsoft Entra ID.
  6. Copy the values for **SCIM URL** and**SCIM Token** when the steps require.
  7. Complete the steps outlined with the instructions document.
