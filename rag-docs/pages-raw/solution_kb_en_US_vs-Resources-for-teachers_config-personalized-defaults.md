# Personalized defaults
If you use multiple devices to present your lessons throughout the day, use the myViewBoard settings page to set up default preferences. Note the following information:
  

  * This requires a ViewSonic account. To create a ViewSonic account, refer to the following article:  
  

    * [Sign up to myViewBoard](https://www.viewsonic.com/solution/kb/3232975)  
  

  * Once set, your preferences will automatically load on any device when you sign in, and ensures that you can start teaching with the same set of preferences on any device.
  * Note that you can only be logged in into one device. Logging in to another device will auto-log you out of all other devices.


  

  

## Access the settings page
To begin setting up your custom preferences, follow the steps below:
  

  1. Launch your browser and open the following site:  
  
  
  

  2. Sign in with your ViewSonic account.   
The ViewSonic dashboard appears.  
  

  3. Go to the **myViewBoard** tile: 
    1. Click the ellipsis to show the menu.
    2. Click **myViewBoard settings**.


  

The myViewBoard Settings page should open. 
  

## Customize session settings
Before presenting your lessons, personalize your myViewBoardâ€™s appearance and behavior ahead of time so every element is ready and supports your lessonâ€™s flow.
  

  

### **Connect online accounts**
Connect one or more cloud service providers that contain the resources you want to use or insert in your lessons. Once connected, the following actions become available: 
  

  * Access images, documents, and other media saved in your cloud drives during your lessons 
  * Save responses, screenshots, and discussions generated during your lessons for review later
  * Set default images for specific myViewBoard features


  

myViewBoard supports all major cloud service providers: Google Drive, OneDrive, DropBox, etc. For more information on how to connect your cloud service account, see the article below:
  

  * [myViewBoard Settings - App & Cloud integration](https://www.viewsonic.com/solution/kb/3184129)


  

  

### **Set default settings**
Specify how myviewboard should look and behave when you login with your account.
  

  * Theme 
  * Session ID and session behavior 
  * Pen colors and size
  * Font size, style, and color


  

After you login, myViewBoard applies all these settings to your current session. Note that you can still change these settings during your session. For more information about the settings screen, see the article below:
  

  * [myViewBoard Settings - Preferences](https://www.viewsonic.com/solution/kb/3184123)


  

  

### **Bookmarks**
Specify web links that should appear on the embedded browser when you login. This is useful when you want to show specific websites that are related your current lessons. You can change these links anytime.
  

For more information on how to add, change or manage these links, see the article below: 
  

  * [myViewBoard Settings - Bookmarks](https://www.viewsonic.com/solution/kb/3184125)


  

  

### **Images**
Specify images that will be loaded as default for various myViewBoard features.
  * Backgrounds
  * Magic Line Pens
  * Shape Pens


  

Note that this feature requires a cloud account. For more information on how to set up default images, see the article below:
  

  * [myViewBoard Settings - Backgrounds](https://www.viewsonic.com/solution/kb/3184125)
  * [myViewBoard Settings - Magic Line Pens](https://www.viewsonic.com/solution/kb/3184127)
  * [myViewBoard Settings - Shape Pens](https://www.viewsonic.com/solution/kb/3184128)


  

  

**Signin required**
Note that the myViewBoard app will only load these settings when you sign in with your account. After you signout, myViewBoard will revert back to the default settings.
