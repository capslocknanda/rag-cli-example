# Admin settings overview
Use the myViewBoard Admin page to customize how myViewBoard behaves in your entity. This page can only be accessed by myViewBoard admins. 
  

**Important**
  

  * Any changes made here only apply to your myViewBoard environment. 
  * You can only access this page if any of the following conditions are true: 
    * You have signed in under a myViewBoard owner account 
    * You have been assigned with the Admin role for the myViewBoard app


  

‚Äç 
  

On this page, myViewBoard admins can use the sidepanel to navigate to various sections where they can perform specific tasks.
  

  

## Entity information
The top of the sidepanel panel shows the entity currently loaded and the owner email. This makes it easy for admins to identify which entity they are making changes to. To load myViewBoard settings for another entity, use the link below. 
  

|  [**Entity list**](https://www.viewsonic.com/solution/kb/en_US/myviewboard-admin-settings/vs-entity-list)  
Load settings for another entity or switch between multiple entities  
---|---  
  

  

## myViewBoard Management
This section allow admins to manage and apply a standard default setup to all myViewBoard app installations within the entity.
  

|  [**Managed Bookmarks**](https://www.viewsonic.com/solution/kb/en_US/vsa-bookmarks)  
Specify global bookmarks that will be displayed for all users  
---|---  
|  [**Cloud Access Control**](https://www.viewsonic.com/solution/kb/en_US/vsa-apps-and-cloud-services)  
Select cloud services that users can use during their myViewBoard sessions  
|  [**Software Instance**](https://www.viewsonic.com/solution/kb/en_US/vsa-software-instances)  
Remotely change the settings of myViewBoard installation  
|  [**Firewall whitelisting**](https://www.viewsonic.com/solution/kb/en_US/vsa-firewall-whitelisting)  
Firewall port and whitelisting requirements to ensure myViewBoard services can run properly  
  

  

## Entity Management
This section allow admins to manage myViewBoard settings that require integration with the ViewSonic Entity account.
  

|  [**User Management**](https://www.viewsonic.com/solution/kb/en_US/vsa-users)  
Select other users who can perform admin duties  
---|---  
|  [**Managed Service Providers**](https://www.viewsonic.com/solution/kb/en_US/vsa-managed-service-providers)  
Specify a Managed Service Provider who can oversee myViewBoard installations  
|  [**NFC Management**](https://www.viewsonic.com/solution/kb/en_US/nfc-management)  
Associate an NFC card with a myViewBoard account registered under your entity  
|  [**Subscriptions**](https://www.viewsonic.com/solution/kb/en_US/vse-subscriptions)  
Opens the Subscription page in the Entity console. Use to review subscription details related to myViewBoard services.  
  

  

## System & Device Management
For more advanced tasks that require monitoring, analysis, and report generation, the myViewBoard Admin page provides links to pages that provide more specialized services:
  

|  [**Insights**](https://www.viewsonic.com/solution/kb/en_US/insights)  
Opens the Insights dashboard. This is an analytics dashboard which provides usage statistics within the selected entity.  
---|---  
|  [**Manager**](https://www.viewsonic.com/solution/kb/en_US/getting-started/overview-manager)  
Opens the myViewBoard Manager console. Use to remotely manage compatible ViewSonic hardware by sending commands.
