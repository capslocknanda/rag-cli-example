# Whiteboard enrollment with Companion
Enroll [instances of Whiteboard](https://www.viewsonic.com/solution/kb/entity-management-1/software-instances) with Whiteboard Enrollment via the Companion App**IT admin** accounts only.
## **Whiteboard for Android & iOS**
End users who install **Whiteboard for Android** or **Whiteboard for iOS** manually can enroll that specific software instance of Whiteboard using Companion's QR code reader.
### On the device
  1. Launch myViewBoard Whiteboard app.
  2. Click **Settings >****software information tab**.
  3. Click **Enroll Software Instance under Entity**.
  4. On the Whiteboard Enrollment Form, click **Enroll via QR Code**. A QR code will display.**  
**


### On Companion
  1. Sign in with your Entity Admin account and tap the 
  2. You will be prompted to select an entity. Ensure you select the correct one, as well as input a Whiteboard name.
  3. Click **Submit**.
  4. A prompt should appear on Whiteboard informing you of successful enrollment.
  5. The device will appear in the **Software Instances** section within **Entity Management** after signing into myviewboard.com.


Access the instance enrollment form under settings.   
IT Admin have the option to enroll an instance via the Companion app QR code reader.
[Delete](https://www.viewsonic.com/solution/kb/en_US/it-support-companion/whiteboard-enrollment)
**WHITEBOARD FOR WINDOWS INSTANCES**
Please note **this enrollment method is currently not available in Whiteboard for Windows**. To learn more, read our [instance management page](https://www.viewsonic.com/solution/kb/entity-management-1/software-instances).
