# Managed Bookmarks
Use the **Managed Bookmarks** page to manage the default bookmarks that appear on the home page of myViewBoard's [Embedded Browser](https://www.viewsonic.com/solution/kb/en_US/create-lessons/embedded-browser).
  

  

  

Each bookmark acts as a quick shortcut to an online resource. Any bookmarks added here will be available to all entity users whenever they access the Embedded Browser home page.
  

  

## How to access
To access this page:
  1. Sign in to 
  2. On the landing page, locate the **myViewBoard** tile.   
Note the ellipsis icon when you hover over.
  3. Click the **myViewBoard admin settings**.
  4. The **myViewBoard admin settings** page appears.
  5. Click **Managed Bookmarks** on the side panel.


  

  

**Important**
  

  * Any changes made here only apply to your myViewBoard environment. 
  * You can only access this page if any of the following conditions are true: 
    * You have signed in under a myViewBoard owner account 
    * You have been assigned with the Admin role for the myViewBoard app

## Add bookmarks
To add a default bookmark, perform the following:
  

  1. On the myViewBoard Admin page, click **Managed Bookmarks**.
  2. On the **Add bookmark** section, provide the following:
    1. **Name**. Specify a unique name for this bookmark
    2. **URL**. Specify a link to a web resource.
    3. **Bookmark icon**. Select the icon that the bookmark will use. By default, this will be the favicon of the target website. Note that some sites will present multiple icons.
  3. Click **Add**.


  

The bookmark is added to the **Entity bookmarks** section. To test the link, click the bookmark icon to load the target website in a new window.
  

Note that bookmarks listed in the **Recommended** section cannot be edited. 
  

  

## Edit bookmarks
If you wish to make changes after the bookmark has been added, click the ellipsis to access the following options:
  

  * **Edit**. Update the Name, URL and bookmark icon, then click **Save**.
  * **Remove**. Removes the bookmark from the list. This action cannot be undone.


  

  

## User bookmarks
Aside from the default bookmarks, entity users can also add their personal bookmarks to link to resources related to the topics they will be discussing. For more details, see the article below:
  * [User bookmarks](https://www.viewsonic.com/solution/kb/account-settings/user-bookmarks)
